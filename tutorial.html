Twitter Analysis tutorial

Hi! This is a work in progress tutorial that will explain you how you can help in analyzing tweets.

The application is really simple. It loads a tweet from Twitter based on a hashtag and asks you to evaluate an emotion: Classify from 0 to 5 the following emotion

All you have to do is to write a value between 0 and 5 and click in the green button that says: Save the answer. Then, the system will load you another tweet and you will be able to analyze the following tweet. That's all!
